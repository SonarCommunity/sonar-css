For readability reasons, there should be one single whitespace between the property colon and its value

Noncompliant Code Example

h1 {
  color:red;         /* Noncompliant: missing whitespace */
}

h1 {
  color:  red;       /* Noncompliant: too many whitespaces */
  color:      green; /* Noncompliant: too many whitespaces */
}

h1 {
  color:
    red;             /* Noncompliant: property and its value not defined on the same line */
}

Compliant Solution

h1 {
  color: red;
}
